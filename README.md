# Dress-Up Doll (Phaser 4 MVP)

A minimal dress-up doll game built with [Phaser 4](https://phaser.io/) and Vite.
All art is generated procedurally at runtime — no image assets required.

## Features

- Doll with 4 clothing categories: hair, top, bottom, shoes
- Cycle variants per category with `<` / `>` buttons
- Randomize button for a full random outfit
- Responsive canvas (FIT scale mode)

## Run

```bash
npm install
npm run dev      # dev server at http://localhost:5173
```

## Build

```bash
npm run build    # production build to dist/
npm run preview  # serve the production build
```

## Quality

```bash
npm run lint     # ESLint
npm run format   # Prettier (write)
```
